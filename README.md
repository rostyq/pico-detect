# pico-detect

Rust library for _Pixel Intensity Comparison-based Object_ (PICO) detection.

## Goals

* [ ] object detection;
* [x] object localization;
* [ ] shape prediction;
* [ ] WebAssembly support (`wasm32-unknown-unknown` target).
